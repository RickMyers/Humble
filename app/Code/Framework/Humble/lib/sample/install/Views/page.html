Some useful locations...

  • &&BASEDIR&&/app/Code/&&PACKAGE&&/&&MODULE&&/Views/&&CONTROLLER&&/Smarty/&&PAGE&&.tpl - location for the main page of your Single Page Application
  • &&BASEDIR&&/app/Code/&&PACKAGE&&/&&MODULE&&/Views/&&CONTROLLER&&/Smarty/index.tpl - this is your "index.html" initial landing page
  • &&BASEDIR&&/app/Code/&&PACKAGE&&/&&MODULE&&/Views/&&CONTROLLER&&/Smarty/404.tpl - this is your 404 "Page Not Found" page
  • &&BASEDIR&&/app/Code/&&PACKAGE&&/&&MODULE&&/Views/&&CONTROLLER&&/Smarty/register.tpl - this page lets others register, you should probably override it
  • &&BASEDIR&&/app/Code/&&PACKAGE&&/&&MODULE&&/Views/&&CONTROLLER&&/Smarty/recover.tpl - the "Forgot My Password" page, starts the recovery process
  • https://humbleprogramming.com - website with extensive documentation and videos for the Humble framework
  • /admin - administration page for your application


Best of luck with everything you are trying to accomplish!
