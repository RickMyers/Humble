Dependencies The installation starts at the command line, and then uses a web form to complete, as a result your web site must be configured (VHOST) and running before the installation takes place.

Furthermore, the following products need to be installed and properly configured prior to attempting install

  • Apache 2.4*
    • Enable Rewrite Module
  • PHP 7.1+
    • Add MongoDB Extension
    • Enable Mysqli Extension
    • Optional
      • Add Memcache Extension
      • Enable SOAP Extension
  • PHP Composer
  • MySQL 5.6+
  • MongoDB 3.4+
  • Optional [optimization]
    • Memcached
  • Optional [support for code deployments]
    • Java JDK 1.8
    • Apache Ant 1.8+
  • Optional [support for developers]
    • Node.js
    • Grunt
